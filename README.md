# ScreenBuddy — Minimal Digital Wellbeing Tracker

ScreenBuddy is a Chrome extension designed to help users understand and manage their screen time through accurate, real-time website usage tracking.

## Features

* Tracks time spent on websites at a domain level
* Real-time tracking using an interval-based background service worker
* Works seamlessly across tabs, reloads, and browsing sessions
* Counts only active and focused usage (no background tracking)
* Clean dashboard with usage insights and visual charts
* Set usage limits and receive notifications when exceeded
* Smart nudges and a subtle “buddy” UI for mindful browsing

## Tech Stack

* JavaScript
* Chrome Extension (Manifest V3)
* Chrome APIs (tabs, storage, idle, notifications)
* Chart.js

## Installation

1. Clone the repository
2. Go to `chrome://extensions/`
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
