# ScreenBuddy — Minimal Digital Wellbeing Tracker

ScreenBuddy is a Chrome extension that helps users understand and manage their screen time through real-time website usage tracking.

## Features

* Tracks time spent on websites (domain-level)
* Works across tabs, reloads, and sessions
* Counts only active and focused usage
* Clean dashboard with visual insights
* Set limits and receive notifications
* Subtle nudges for mindful browsing

## Tech Stack

* JavaScript
* Chrome Extension (Manifest V3)
* Chrome APIs (tabs, storage, idle, notifications)
* Chart.js

## Installation

1. Clone the repository
2. Go to `chrome://extensions/`
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
